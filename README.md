### How to Clone this Repository using Git Bash
1. Go to my dotfiles repository located here: https://github.com/MahalaCovey/dotfiles
2. Click the green "<> Code" button and copy the web URL
3. Open Git Bash and type "git clone <web URL>"
4. Hit the "Enter" button on your computer

### How to Utilize this Repository
- Read this README.md file  
- Read the README.md files in each directory before using their files
- Read the comments in each script before running them to gain an understanding of what they do
- Run each bash script with the "bash <filename>" command
